Ejercicio 2: Funciones de orden superior¶

Para cada uno de las siguientes pares de funciones decir cual es Big O de la función que se representa por el par.

  • f(n) = (n^2 - n)/2 , g(n) = 6n
  • f(n) = n + 2 $\sqrt n$ , g(n) = n2
  • f(n) = n + n log n , g(n) = n $\sqrt n$
  • f(n) = n^2 +3n +4 , g(n) = n^3
  • f(n) = n + log n , g(n) = $\sqrt n$
In [6]:
# Configuración de la gráficas
n = np.arange(1, 100)
fn1 = (n**2 - n)/2
gn1 = 6*n
fn2 = n + 2*np.sqrt(n)
gn2 = n**2
fn3 = n + n*np.log(n)
gn3 = n*np.sqrt(n)
fn4 = n**2 + 3*n + 4
gn4 = n**3
fn5 = n + np.log(n)
gn5 = np.sqrt(n)

Solución¶

In [8]:

Gráficamente podemos concluir que para la primera pareja la función que mas crece es f(n).
